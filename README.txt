Ввод: Считывает строку из input.txt
Клаузы разделены пробелами, символы внутри клауз вводятся слитно.
Отрицание обозначается заглавной буквой.
Например, {{¬x1,¬x2},{x2,x3,¬X4,¬x1}} == X1X2 x2x3X4X1
---------------------------------
Вывод: записывает в файл output.txt
Первая строка - это список переменных, представленных в виде x1 x2 x3.
Вторая строка - это возможная интерпретация переменных, при который формула выполняется.
Если данные не выданы - никакая интерпретация невозможна.
(-1 означает, что переменная может иметь любое логическое значение: 0 или 1)
